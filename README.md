# IT Support Notes

This repository contains structured notes, procedures, and troubleshooting logs focused on IT support and networking fundamentals.

The purpose of this repository is to document hands-on learning, problem-solving approaches, and common scenarios encountered in entry-level IT support roles.

---

## Topics covered

- Basic computer hardware and peripherals
- Operating system fundamentals (Windows / Linux basics)
- Networking fundamentals (IP addressing, DNS, DHCP, basic routing)
- Troubleshooting methodology and documentation
- User support scenarios and common issues
- Command-line basics and diagnostic tools

---

## Repository structure

This repository is organized by topic. Each section contains clear notes, step-by-step procedures, and troubleshooting examples.

---

## Disclaimer

This repository is for educational and practice purposes. It reflects hands-on learning and simulated or real-world troubleshooting scenarios in a training context.
